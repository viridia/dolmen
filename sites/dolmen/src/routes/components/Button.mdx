# Button

Buttons help people initiate actions, from sending an email, to sharing a document, to liking a post.

## Button Colors
Dolmen supports a small number color variants out of the box, although this can be extended
with custom styles.

**Primary** buttons are intended for calls to action. There should only be one primary button
within any page section or modal dialog.

**Danger** buttons are used to warn the user that the action is irreversible and may have serious
consequences.

## Button Icon Colors
Icons can be used as children of a `Button` component. The CSS variable `--icon-color` is set to
the icon color corresponding to the button's color scheme and theme. If you are using SVG icons,
you can add `fill="var(--icon-color)"` to your SVG elements, and they will automtically pick up the
icon color from the button.

element.
