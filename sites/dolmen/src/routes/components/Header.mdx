# Header

`Header` renders an HTML heading element. The main reason for using `Header` as opposed to
a plain HTML header element is control over size. Dolmen stylesheets don't define styles
for vanilla HTML elements, but only for Dolmen components.

<DemoSection>
  <Header>default header style</Header>
  <Header as="h1">h1</Header>
  <Header as="h2">h2</Header>
  <Header as="h3">h3</Header>
  <Header as="h4">h4</Header>
  <Header as="h5">h5</Header>
</DemoSection>
