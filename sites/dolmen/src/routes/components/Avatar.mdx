# Avatar

`Avatar` displays a small image that identifies a user or entity.

## Avatar Radius

The `radius` property determines the roundness of the avatar corners. Setting the radius to `xl`
makes the avatar circular.

<DemoSection>
  <Group gap="md">
    <Avatar radius="xs" color="#00ff88">xs</Avatar>
    <Avatar radius="sm" color="#00cc88">sm</Avatar>
    <Avatar radius="md" color="#008888">md</Avatar>
    <Avatar radius="lg" color="#004488">lg</Avatar>
    <Avatar radius="xl" color="#0044cc">xl</Avatar>
  </Group>
</DemoSection>
<SourceCode>
```tsx
<>
  <Avatar radius="xs" color="#00ff88">xs</Avatar>
  <Avatar radius="sm" color="#00cc88">sm</Avatar>
  <Avatar radius="md" color="#008888">md</Avatar>
  <Avatar radius="lg" color="#004488">lg</Avatar>
```

</SourceCode>

## Avatar properties

<PropsTable>
  <PropsItem
    name="src"
    description="Image URL."
    type="string"
  />
  <PropsItem
    name="crossOrigin"
    description="Cross-origin attribute to add to the image element."
    type="JSX.HTMLCrossorigin"
  />
  <PropsItem
    name="title"
    description="Image 'title' property."
    type="string"
  />
  <PropsItem
    name="color"
    description="Fill color of the avatar (when no image is displayed)."
    type="CSS color expression"
  />
  <PropsItem
    name="colorHash"
    description="String used to generate a random color (when no image is displayed)."
    type="string"
  />
  <PropsItem
    name="size"
    description="Avatar size."
    type={
      <UnionType values={['xl', 'lg', 'md', 'sm', 'xs']} />
    }
  />
  <PropsItem
    name="radius"
    description="Indicates how 'rounded' the avatar should appear."
    type={
      <UnionType values={['xl', 'lg', 'md', 'sm', 'xs']} />
    }
  />
</PropsTable>

`Avatar` also accepts all of the standard attributes for a `div` element.
