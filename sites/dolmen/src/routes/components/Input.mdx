# Input

The `Input` component renders an HTML <HTMLTag>input</HTMLTag> element. It has a frw additional
features, such as the ability to add icons (called "adornments") inside the field box.

### Properties

<PropsTable>
  <PropsItem
    name="adornLeft"
    description="Adornments to place on the left side of the input field"
    type="JSX.Element"
  />
  <PropsItem
    name="adornRight"
    description="Adornments to place on the right side of the input field"
    type="JSX.Element"
  />
  <PropsItem
    name="innerClass"
    description="CSS class to add to the inner 'input' element. This is in contrast to 'class', which is applied to the outer wrapper."
    type="string"
  />
  <PropsItem
    name="innerClassList"
    description="CSS classList to add to the inner 'input' element. This is in contrast to 'classList', which is applied to the outer wrapper."
    type="object"
  />
  <PropsItem
    name="size"
    description="Controls the height of the component."
    type={
      <UnionType values={['xl', 'lg', 'md', 'sm', 'xs']} />
    }
  />
  <PropsItem
    name="round"
    description="Whether the field should have rounded corners."
    type="boolean"
  />
</PropsTable>

In addition, `Input` accepts the standard HTML attributes for an <HTMLTag>input</HTMLTag> element.
