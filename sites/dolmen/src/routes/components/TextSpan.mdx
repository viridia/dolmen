# TextSpan

`TextSpan` simply renders a span of text in a particular style.

## TextSpan sizes

<DemoSection>
  <Stack gap="xl">
    <TextSpan>Default size</TextSpan>
    <TextSpan size="xs">size="xs"</TextSpan>
    <TextSpan size="sm">size="sm"</TextSpan>
    <TextSpan size="md">size="md"</TextSpan>
    <TextSpan size="lg">size="lg"</TextSpan>
    <TextSpan size="xl">size="xl"</TextSpan>
  </Stack>
</DemoSection>
<SourceCode>
```tsx
<TextSpan>Default size</TextSpan>
<TextSpan size="xs">size="xs"</TextSpan>
<TextSpan size="sm">size="sm"</TextSpan>
<TextSpan size="md">size="md"</TextSpan>
<TextSpan size="lg">size="lg"</TextSpan>
<TextSpan size="xl">size="xl"</TextSpan>
```
</SourceCode>

## Element type

The `as` property allows the text span to be rendered as different element types:

<DemoSection>
  Text with<TextSpan as="sup">superscript</TextSpan> and <TextSpan as="sub">subscript</TextSpan>
</DemoSection>
<SourceCode>
```tsx
  Text with<TextSpan as="sup">superscript</TextSpan> and <TextSpan as="sub">subscript</TextSpan>
```
</SourceCode>

`TextSpan` also accepts all of the standard attributes for a `span` element.
