# DiscloseButton

A `DiscloseButton` is used to hide or reveal a section of content.

<DemoSection>
  {() => {
    const [expand, setExpand] = createSignal(false);
    return <DiscloseButton open={expand()} onClick={() => setExpand(v => !v)} />;
  }}
</DemoSection>
<SourceCode>
```tsx
const [expand, setExpand] = createSignal(false);
return (<DiscloseButton open={expand()} onClick={() => setExpand(v => !v)} />);
```
</SourceCode>

## DiscloseButton properties

<PropsTable>
  <PropsItem name="open" description="Open or close state of the button." type="boolean" />

In addition, the `DiscloseButton` element accepts all of the standard attributes of the HTML
<HTMLTag>button</HTMLTag> element.
