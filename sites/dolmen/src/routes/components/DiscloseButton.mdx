# DiscloseButton

A `DiscloseButton` is used to hide or reveal a section of content.

<DemoSection>
  {() => {
    const [expand, setExpand] = createSignal(false);
    return <DiscloseButton open={expand()} onClick={() => setExpand(v => !v)} />;
  }}
</DemoSection>
