# Select

`Select` allows a user to select one option from a list of options.

The Select component is generic - the type argument is the type of option value, which can
be a number, string, enumeration value, or any value that is comparable via `===`.

</DemoSection>
<SourceCode>
```tsx
const [selected, setSelected] = createSignal(-1);

return (
  <Select
    selected={selected()}
    onSelect={setSelected}
    placeholder="Select a number..."
    options={[
      {
        value: 1,
        label: 'One',
      },
      {
        value: 2,
        label: 'Two',
      },
      {
        value: 3,
        label: 'Three',
      },
      {
        value: 4,

      },
      {
        value: 5,
        label: 'Five',
      },
      {
        value: 100,
        label: 'A gigantic googleplex of infinite multitudes',
      },
    ]}
  />
);
```
</SourceCode>

## Option objects

The `Select` component takes a list of options, which have the following type:

<SourceCode>
```tsx
interface Option<T> {
  /** The value of the option */
  value: T;

  /** Label to be displayed in the menu. */
  label: JSX.Element;

  /** Whether the option should be displayed as disabled. */
  disabled?: boolean;
}
```
</SourceCode>
