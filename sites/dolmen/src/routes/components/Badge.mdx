# Badge

A `Badge` is used to indicate that a group of buttons are related or part of a set.

## Badge Colors

The `color` property of a `Badge` can be set to any CSS color expression. The component
automatically selects a high-contrast text color to go with the specified fill color.

<DemoSection>

## Badge Radius
The `radius` property determines the roundness of the badge corners. Setting the radius to `xl`
converts the badge into a pill shape.

  <Group gap="md">
