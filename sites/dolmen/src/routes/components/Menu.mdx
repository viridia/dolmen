# Menu

Dropdown menus display a list of actions or selectable options. The menu button, when triggered,
displays a floating overlay element containing a list of available choices. Choosing an option
should ideally commit the option and close the menu.

Menus generally represent "actions" or "states" - that is, choosing the menu performs the action
or applies the specified state. Menus should not be used to select between a mutually exclusive
set of items - instead, use the `Select` component for that.

Menus are designed to support keyboard navigation. Arrow keys can be used to open the menu
if the menu button has focus, and can be used to navigate between the items once opened.

## Menu elements

A Menu contains several elements:

* A `Menu.Button` which opens the menu.
* A `Menu.List` which contains one or more items. This is the list of items that will be displayed
  insude the menu.

Menu lists can contain one or more of the following:

* `Menu.Item` - a clickable item.
* `Menu.ItemCheckBox` - a clickable item, with a checkmark, that toggles between two states.
* `Menu.ItemRadio` - selectable item which is part of a mutually exclusive set.
* `Menu.Divider` - a separator which puts a dividing line between items.
