# FormField

The `FormField` component wraps a form input, such as a text input field, and adds support for
validation messages and additional decorations.

The `error` message and other status messages are intended to display field validation errors
and other messages that pertain to the content of an individual field. For errors which concern
the entire form (such as a server error response), it is better to use an `Alert`.

### Properties

Note that the properties `error`, `warning`, `info`, and `success` are exclusive - only a single
message can be displayed.
