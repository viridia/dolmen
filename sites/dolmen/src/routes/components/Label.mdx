# Label

The `Label` component simply creates an HTML <HTMLTag>label</HTMLTag>. The only different from
a plain label is that it supports Dolmen <A href="../../start/flex">flex props</A>.

In most cases, a `FormField` component will be a better choice for associating a label with
a form input. However, there may be cases
where you want to associated a label with a form input component that is not a text input.
