# ScrollArea

A `ScrollArea` is a layout component arranges its child elements verticall and allows them
to be scrolled.

### Properties

The `ScrollArea` component has no special properties of its own. However, like all layout components,
it accepts both the standard Dolmen <A href="../../start/flex">flex props</A> as well
as the standard HTML element properties.
