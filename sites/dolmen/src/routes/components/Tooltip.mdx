# Tooltip

A tooltip displays additional information about an element when the user hovers over it. Tooltips
can be added to any element by wrapping that element with the `Tooltip` component.

**Note**: A much easier way to create tooltips is to initialize a global
<A href="../../hooks/createTooltipWatcher">tooltip watcher</A>. The `Tooltip` component is
primarily meant for cases where you need precise control over the content and style of a tooltip.

<DemoSection>
  <Tooltip
    content={
      <Stack>
        <Header>Tip Title</Header>
        <div>Once upon a time, in a land very far away, there lived a young man.</div>
      </Stack>
    }
  >
    <Button>Tooltip Example</Button>
  </Tooltip>
</DemoSection>
<SourceCode>
```tsx
<Tooltip
  content={
    <Stack>
      <Header>Tip Title</Header>
      <div>Once upon a time, in a land very far away, there lived a young man.</div>
    </Stack>
  }
>
  <Button>Tooltip Example</Button>
</Tooltip>
```
</SourceCode>

The content of a tooltip can be either plain text or JSX elements.
