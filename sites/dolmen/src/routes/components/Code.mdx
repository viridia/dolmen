# Code

`Code` renders a <HTMLTag>code</HTMLTag> element.

<DemoSection>
  <Code>const [selected, setSelected] = createSignal(false);</Code>
</DemoSection>
<SourceCode>
```tsx
<Code>const [selected, setSelected] = createSignal(false);</Code>
```
</SourceCode>

`Code` by itself does not do syntax highlighting, as this would require adding a dependency
on [prism.js](https://prismjs.com/) or [highlight.js](https://highlightjs.org/). However, it's
fairly easy for the application to add syntax highlighting using these libraries.

## Code.Block

To render a block of code, use `Code.Block`. This wraps the <HTMLTag>code</HTMLTag> element in
a <HTMLTag>pre</HTMLTag> element.

<DemoSection>
  <Code.Block>
    const [selected, setSelected] = createSignal(false);

    return selected();
  </Code.Block>
</DemoSection>
<SourceCode>
```tsx
<Code.Block>
  const [selected, setSelected] = createSignal(false);
  return selected();
</Code.Block>
```
</SourceCode>
