# ToggleSwitch

Toggle switches can be used to turn an option on or off.

<DemoSection>
  {() => {
    const [checked, setChecked] = createSignal(false);
    return (
      <ToggleSwitch checked={checked()} onClick={() => setChecked(v => !v)} >
        I like Chocolate!
      </ToggleSwitch>
    );
  }}
</DemoSection>

## ToggleSwitch properties

Toggle switches accept all of the standard properties for an HTML checkbox element.
